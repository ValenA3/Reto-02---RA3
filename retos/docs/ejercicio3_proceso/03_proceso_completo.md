# PROCESO COMPLETO

**Introducción**
El objetivo de este reto ha sido preparar un USB booteable con Ventoy que tenga varias distribuciones Linux, y instalar al menos una de ellas en un pc real, documentando todo el proceso, los intentos realizados y los posibles errores

**Preparación del USB**
Primero preparamos un USB con Ventoy, que permite meter varias ISOs sin tener que formatear cada vez
Después comprobamos que el USB arrancaba bien y que aparecía el menú de Ventoy con todas las opciones

**Decisiones tomadas**
Entramos en la BIOS y cambiamos el orden de arranque para que arrancara desde el USB.
Primero probamos con Puppy Linux y al principio funcionaba bien. Pero al día siguiente Ventoy dejó de funcionar y no arrancaba.
Entonces usamos un USB de un compañero que estaba hecho con Rufus, y ahí sí funcionó correctamente. Con ese USB decidimos instalar antiX Linux
En otro ordenador probamos a instalar MX Linux y también funcionó sin problemas

**Decisiones tomadas**
Elegimos distribuciones ligeras porque el ordenador es antiguo.
Probamos varias ISOs por si alguna fallaba.
Cuando Ventoy dejó de funcionar, buscamos otra solución usando Rufus.
Al final elegimos antiX Linux porque funcionaba bien y era estable.
