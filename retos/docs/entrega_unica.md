**Preparación del USB**
Primero descargamos Ventoy y lo ejecutamos en el ordenador.
Después seleccionamos el USB y lo instalamos. En ese momento el USB se formateó y se preparó automáticamente para poder arrancar sistemas desde él.

Copia de las ISOs:
Una vez instalado Ventoy, simplemente copiamos los archivos ISO dentro del USB como si fuera una carpeta normal.
Yo meti antiX Linux, Bodhi Linux y Puppy Linux

Para comprobar que todo funcionaba bien, arrancamos el ordenador desde el USB.
Al iniciar, apareció el menú de Ventoy donde se veían todas las ISOs que se habian copiado

**Instalación en el equipo**
Acceso a BIOS<br>
Cambio de orden de arranque (USB primero)<br>
